## About react-gm

愿景：致力于快速搭建项目，像搭积木一样

## Install

`npm install react-gm -D`

库文件和样式文件在dist下

## 依赖

使用前请保证以下依赖存在
- react react-dom
- [react-bootstrap](http://react-bootstrap.github.io/) 后续会慢慢脱离react-bootstrap
- classnames
- underscore
- moment
- [gm-bootstrap](https://github.com/gmfe/gm-bootstrap) css主题
